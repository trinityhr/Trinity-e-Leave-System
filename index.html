🔔 LINE Notify Token

แจ้งเตือนผู้บริหารทาง LINE เมื่อมีพนักงานยื่นลาใหม่
วิธีสร้าง Token ฟรี:
1. ไปที่ notify-bot.line.me → เข้าสู่ระบบด้วย LINE
2. กด "Generate token" → ตั้งชื่อ → เลือก กลุ่ม LINE ผู้บริหาร (หรือ 1:1 กับตัวเอง)
3. กด Generate → คัดลอก token มาวางด้านล่าง
4. เพิ่ม LINE Notify เป็นเพื่อนในกลุ่ม LINE ที่เลือกด้วย

🔐 รหัสผ่านหน้าอนุมัติ

ล็อกหน้าอนุมัติและตั้งค่า ไม่ให้พนักงานทั่วไปเข้าได้

📧 แจ้งอีเมลพนักงาน (EmailJS)

ส่งอีเมลแจ้งพนักงานอัตโนมัติเมื่ออนุมัติหรือปฏิเสธ — ฟรี 200 อีเมล/เดือน
วิธีตั้งค่า EmailJS ฟรี:
1. ไปที่ emailjs.com → Sign up ฟรี
2. Add New Service → เลือก Gmail → เชื่อมบัญชี Gmail → จด Service ID
3. Email Templates → Create New Template → ตั้งค่า Subject/Body ตามชอบ → จด Template ID
   (ใช้ variable: {{to_email}} {{name}} {{status}} {{leave_type}} {{start}} {{end}})
4. Account → จด Public Key
5. นำทั้ง 3 อย่างมากรอกด้านล่าง
